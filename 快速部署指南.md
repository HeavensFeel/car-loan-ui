# 车贷小程序进件系统 - 快速部署指南

## 🚀 部署到 Netlify（推荐 - 最快方式）

由于 Vercel 需要复杂的配置，我们推荐使用 **Netlify**，它更简单、更快！

### 第一步：打开 Netlify 网站

访问：https://app.netlify.com/drop

### 第二步：拖拽上传

1. 打开 Finder（我已帮你打开项目文件夹）
2. 将整个 **`car-loan-ui`** 文件夹拖到 Netlify 的上传区域
3. 等待上传完成（约5-10秒）

### 第三步：获取公网地址

部署完成后，Netlify 会自动生成一个公网地址，例如：
```
https://car-loan-ui-demo.netlify.app
```

**把这个地址分享给你的团队伙伴即可！**

---

## 📱 部署完成后

你可以分享以下链接：

## 💡 自定义域名（可选）

在 Netlify 后台的 Site settings 中，可以将随机域名修改为：
```

```
