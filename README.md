# 车贷小程序进件系统 - 高保真UI设计稿

## 项目概述
完整的车贷小程序进件系统高保真UI设计，包含全业务流程页面。

## 页面清单

### 模块1：工作台 & CRM
- **page-home.html** - 3个屏幕
  - 工作台首页
  - CRM·线索清单
  - CRM·线索详情

### 模块2：车贷进件流程
- **page-loan.html** - 4个屏幕
  - 扫码进件
  - 进件资料填写
  - 上传文件
  - 大数据预审

### 模块3：审批流程
- **page-approval.html** - 5个屏幕
  - 进件审批进度
  - 面签预约
  - 面签完成
  - GPS安装
  - 放款成功

### 模块4：贷后管理
- **page-post-loan.html** - 4个屏幕
  - 还款提醒
  - 还款登记
  - 逾期详情
  - 统计报表

### 模块5：管理功能
- **page-admin.html** - 4个屏幕
  - 机构管理
  - 账号管理
  - 产品管理
  - 个人中心

## 设计系统
- **主色调**: 蓝色 #1677FF
- **辅助色**: 绿色(成功) #52C41A、橙色(警告) #FAAD14、红色(危险) #FF4D4F
- **设计风格**: 现代简约、卡片式布局、渐变色彩
- **适配**: 375px 标准手机屏幕

## 功能模块对照

### CRM模块
✅ 线索清单 - 多维度筛选、状态标识
✅ 线索处理 - 详情查看、跟进记录、转进件
✅ 线索统计 - 状态分布、转化分析

### 车贷模块
✅ 预审 - OCR识别、大数据风控、信用评分
✅ 进件 - 产品选择、参数配置、资料上传
✅ 审批 - 流程进度、审批意见、终审决策
✅ 面签 - 预约管理、文件签署、面签登记
✅ GPS安装 - 设备分配、安装登记、照片上传
✅ 放款 - 放款确认、还款计划、成功通知

### 贷后模块
✅ 还款提醒 - 到期提醒、逾期监控、催收管理
✅ 还款登记 - 实收登记、凭证上传、记录归档
✅ 逾期信息 - 逾期详情、催收记录、联系管理

### 管理功能
✅ 机构管理 - 分行信息、员工配置、状态管理
✅ 账号管理 - 员工账号、权限配置、登录状态
✅ 产品管理 - 产品配置、利率设置、使用统计

## 技术栈
- 纯HTML + CSS
- 无需JavaScript交互（展示用）
- 响应式布局
- 375px手机壳模拟

## 使用方法
直接用浏览器打开任一HTML文件即可查看对应模块的高保真设计稿。

所有页面遵循统一的设计系统和交互规范，可直接用于开发参考或客户演示。
