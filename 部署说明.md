# 🚀 部署到公网 - 操作指南

## 方案一：使用 Vercel（推荐，最快）

### 1️⃣ 在 GitHub 上创建仓库
1. 访问 https://github.com/new
2. 仓库名：`car-loan-ui`（或其他你喜欢的名字）
3. 选择 Public（公开）或 Private（私有）
4. 点击 **Create repository**

### 2️⃣ 推送代码到 GitHub
在终端执行以下命令：

```bash
cd /Users/xuzekun/WorkBuddy/20260401092557/car-loan-ui

# 添加远程仓库（替换 YOUR_USERNAME 为你的GitHub用户名）
git remote add origin https://github.com/YOUR_USERNAME/car-loan-ui.git

# 推送代码
git branch -M main
git push -u origin main
```

### 3️⃣ 在 Vercel 上部署
1. 访问 https://vercel.com/signup
2. 使用 GitHub 账号登录
3. 点击 **"Add New..."** → **"Project"**
4. 选择刚才创建的 `car-loan-ui` 仓库
5. 点击 **"Deploy"**

### 4️⃣ 获取访问地址
部署完成后（约30秒），Vercel 会提供一个公开的 HTTPS 地址，例如：
```
https://car-loan-ui.vercel.app
```

**✅ 完成！把这个地址分享给团队伙伴即可！**

## 方案二：使用 GitHub Pages（免费，稳定）

### 1️⃣ 推送代码到 GitHub
同上，先推送到 GitHub 仓库

### 2️⃣ 启用 GitHub Pages
1. 进入仓库 → **Settings**（设置）
2. 左侧菜单找到 **Pages**
3. Source 选择：`Deploy from a branch`
4. Branch 选择：`main` → `/ (root)`
5. 点击 **Save**

### 3️⃣ 等待部署
等待1-2分钟，GitHub Pages 会自动构建并部署

### 4️⃣ 获取访问地址
访问地址格式：
```
https://YOUR_USERNAME.github.io/car-loan-ui/
```

## 方案三：使用 Netlify（最简单，拖拽部署）

### 1️⃣ 访问 Netlify
https://app.netlify.com/drop

### 2️⃣ 拖拽上传
直接将整个 `car-loan-ui` 文件夹拖到页面中

### 3️⃣ 获取访问地址
Netlify 会立即生成一个随机地址，例如：
```
https://silly-kalam-123456.netlify.app
```

可以在 **Site settings** 中修改为自定义域名
