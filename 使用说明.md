# 车贷小程序进件系统 - 高保真UI设计稿

## 📂 文件说明

本设计稿采用 **5个HTML文件** 承载 **20个页面**，每个HTML文件内部通过页面标签切换显示不同页面。

### 📱 页面导航

#### **1. page-home.html** - 工作台 & CRM模块（3屏）
- **标签1**: 工作台首页 - 业绩概览、快捷功能、待办事项
- **标签2**: CRM·线索清单 - 客户线索列表、筛选功能
- **标签3**: CRM·线索详情 - 客户信息、跟进记录、转化进件

#### **2. page-loan.html** - 进件流程（4屏）
- **标签1**: 扫码进件 - OCR识别、身份证/VIN码扫描
- **标签2**: 进件资料填写 - 产品选择、贷款参数、月供计算
- **标签3**: 上传文件 - 资料上传、进度展示、文件管理
- **标签4**: 大数据预审 - 征信查询、风控检测、预审结论

#### **3. page-approval.html** - 审批流程（5屏）
- **标签1**: 进件审批进度 - 流程时间轴、当前状态、审批意见
- **标签2**: 面签预约 - 时间地点选择、材料清单、预约确认
- **标签3**: 面签完成 - 文件签署、签约确认
- **标签4**: GPS安装 - 设备信息、安装登记、照片上传
- **标签5**: 放款成功 - 放款信息、还款计划、后续事项

#### **4. page-post-loan.html** - 贷后管理（4屏）
- **标签1**: 还款提醒 - 逾期监控、催收管理、联系方式
- **标签2**: 还款登记 - 实收登记、还款凭证、备注信息
- **标签3**: 逾期详情 - 逾期信息、催收记录、联系管理
- **标签4**: 统计报表 - 核心数据、进件漏斗、月度趋势

#### **5. page-admin.html** - 管理功能（4屏）
- **标签1**: 机构管理 - 分行信息、员工配置、状态管理
- **标签2**: 账号管理 - 员工账号、权限配置、登录状态
- **标签3**: 产品管理 - 产品配置、利率设置、使用统计
- **标签4**: 个人中心 - 个人信息、业绩数据、系统设置

## 🎨 设计系统

## 🚀 使用方法

### 方式1: 直接打开文件
在浏览器中打开任意 `.html` 文件：
- Mac: 双击文件或右键"打开方式" → "Safari/Chrome"
- Windows: 双击文件即可

### 方式2: 使用本地服务器
```bash
# 进入项目目录
cd /Users/xuzekun/WorkBuddy/20260401092557/car-loan-ui

# 启动Python简易服务器
python3 -m http.server 8000

# 在浏览器访问
open http://localhost:8000/page-home.html
```

### 页面导航切换
每个HTML文件顶部有 **页面导航标签**，点击即可切换查看该文件内的所有页面。
